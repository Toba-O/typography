Typography - ZEIT UI

You can get the Typography style from ZEIT Design here. And support Dark Mode. (add class="dark-theme" to html tag to switch dark mode.)

Use by NPM:

import '@zeit-ui/themes'
import '@zeit-ui/typography'

Use by CDN:

<link href="https://cdn.jsdelivr.net/npm/@zeit-ui/typography@latest/umd.css" rel="stylesheet">
